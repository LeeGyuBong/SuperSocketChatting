# SuperSocketStudy

목표   
```
1. 다중 클라이언트가 서버에 접속해서 서로간 대화가 가능한 채팅 서버 구현 - 완료   
2. 채팅 외 기능을 추가하여 채팅은 SuperSocket을 사용한 소켓 통신, 그 외는 Http 통신으로 동작하도록 구현
```

* * *
클라이언트 사용버전   
```
.NET 6.0
SuperSocket.ClientEngine v0.10.0
```

서버 사용버전
```
.NETFramework v4.8   
SuperSocket v1.6.6.1
```

사용 기타 라이브러리   
```
MessagePack v2.5.187
```
